This directory contains the results from Missense3D. In order
to view the results, simply open the file index.html in your
local browser.

Please note that the coordinates for the wildtype and mutant structures
can be found in the directory "mutant_structures". Both structures have
been repacked using the SCWRL4 program.  
Your wildtype structure is labelled
     PDB-[pdb code or file name]-[Chain ID].sc.pdb
Your mutant structure is labelled
     PDB-[pdb code or file name]-[Chain ID]-[Mutant residue ID and name].sc.pdb
